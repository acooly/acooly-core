Acooly framework-CSS/JS

acooly.format.js

1.文件大小格式化

根据输入的byte大小，自动格式化为b,k,m,g等可读显示方式
// easyui的datagrid列定义
<th field="fileSize" formatter="fileSizeFormatter">文件大小</th>
// javascript
$.acooly.format.fileSize(101212);
byte

2.金额货币格式化

传入金额，单位（元，分）和符号($,￥), 自动格式化为2位小数点的货币显示方式，支持四舍五入
// easyui的datagrid列定义
<th field="amount" formatter="moneyFormatter">金额</th>
// javascript
$.acooly.format.money($('#amount').val(),true,'￥')
分

3.日期时间格式化

。。。
// easyui的datagrid列定义
<th field="createTime" formatter="dateTimeFormatter">创建时间</th>
<th field="createTime" formatter="dateFormatter">生日</th>
// javascript
$.acooly.format.date(new Date(),'yyyy-MM-dd HH:mm:ss');
$.acooly.format.date('2016-08-90 12:12:12','yyyy-MM-dd');
$.acooly.format.date('Sun Jul 10 2016 01:46:47 GMT+0800 (CST)','yyyy-MM-dd');
'yyyy-MM-dd HH:mm:ss'

4.时长格式化

在记录诸如执行时长，通话时长时，我们一般会按毫秒或秒的方式记录。但是并不方便人类直观查看，本格式化用于格式时长数据为: x天12:12:12 的格式
// easyui的datagrid列定义
<th field="executeTimes" formatter="millisecondFormatter">执行时长</th>
<th field="talkTime" formatter="secondFormatter">通话时长</th>
// javascript
$.acooly.format.timespan(2000);
$.acooly.format.timespan(2000,"ms");

4.长文本格式化

用于当行显示（如：列表中）长文本数据。
// easyui的datagrid列定义
<th field="subject" formatter="contentFormatter">简介</th>
// javascript
$.acooly.format.content("在记录诸如执行时长，通话时长时，我们一般会按毫秒或秒的方式记录。");

5.链接格式化

用于对URL链接的格式化。
// easyui的datagrid列定义
<th field="homepage" formatter="linkFormatter">简介</th>
// javascript
$.acooly.format.link("http://acooly.cn");
